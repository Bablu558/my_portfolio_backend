# Portfolio Backend (MERN Stack)

This repository contains the backend of my personal **Portfolio & Blog Platform**.
It is built using **Node.js, Express.js, and MongoDB** and follows a clean,
scalable, and production-ready architecture.

The project includes authentication, blog management, email services,
file uploads, and secure API handling.

This backend was successfully deployed on **AWS EC2 (Ubuntu)** as part of
cloud and DevOps learning.

### Deployment Highlights:

- EC2 instance launched and configured
- SSH access using key-pair authentication
- Node.js & dependencies installed on Linux server
- Environment variables configured securely
- Backend process managed using **PM2**
- MongoDB Atlas connected remotely
- Required ports opened using AWS Security Groups
- Application verified via public IP
